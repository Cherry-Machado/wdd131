Conheça Dona Elvira

Dona Elvira não é apenas uma costureira; ela é uma verdadeira "encantadora de tecidos"! Com uma paixão pela costura que acende cada linha, ela dá vida aos seus sonhos de vestuário. Esqueça os ajustes padronizados; o serviço personalizado de Dona Elvira garante que cada peça seja feita sob medida com perfeição, só para você.

Sua jornada começou aos 14 anos, quando decidiu se tornar uma profissional em alta costura. Agora, com décadas de experiência, Dona Elvira dominou a arte dos ajustes, tornando-se a especialista preferida de quem valoriza qualidade e estilo. Seja uma simples bainha ou um vestido sob medida para uma ocasião especial, Dona Elvira é o seu destino final para tudo relacionado à costura. ¡Prepare-se para desfilar em roupas que vestem como uma luva!
